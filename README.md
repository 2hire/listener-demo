# listener-demo

## Set up

### Install dependencies

	npm install

### Compile schemas

	npm run compile-schemas
	
## Run

### Build

	npm run build
	
### Start

	npm run start

## Try

### Hello world

	curl localhost:8080/hello

### Listener

Please follow the Webhooks listener tutorial on the dev portal:
https://developer.2hire.io/docs/webhooks-listener#run
